Menu Ajuda

Introdução

O Projeto é desenvolvido utilizando a biblioteca WxWidgets e C++. Possui as ferramentas principais e um conjunto de cores para possibilitar que o usuário logue no sistema e cadastro de usuário com logs de segurança.

Menu

O menu é constituido de 3 itens, e os acessos são conforme a estrutura hierarquica
  • Arquivo
    • novo
    • abrir
    • salvar
    • fechar
  • Acesso com Novo usuario
  • Ajuda e item sobre Sobre.

Descrição Detalhada

Novo

Abre uma mensagem de novo arquivo

Abrir

Abre uma janela de pesquisa de arquivo para selecionar um arquivo.

Salvar

Abre uma caixa de mensagem informando que um arquivo foi salvo

Fechar

Finaliza o programa

Novo Usuario

Uma janela com todos os seguintes campos necessário ao cadastro de usuario

  1. nome - armazena o nome do usuário
  2. usuario - armezana um texto para discriminar o usuário
  3. senha - armazena a senha do usuário
